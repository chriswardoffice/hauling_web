# Hauling Web App

A web application for hauling logistics, built with Rust (via Cargo) and featuring a quotes database.

## Setup

- Install Rust: `curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh`
- Run: `cargo run`

## Features

- Sick features

## Database

Uses `quotes.db` for storing quotes (SQLite).
